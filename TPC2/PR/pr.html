    • Abstract:

    Foi-nos proposta a realização de um trabalho de casa que tem como objetivo criar uma ontologia usando um dataset disponibilizado em xml sobre música.
    Em primeiro lugar, foi criada a estrutura inicial da ontologia usando a ferramenta Protégé. Neste caso, foram definidas as classes: Obra, Compositor, Instrumento e Partitura. De seguida, foram criados os data properties e object properties necessários assim como alguns indivíduos.
    Em segundo lugar, foi criado um ficheiro XSLT que teve como objetivo converter o dataset XML num ficheiro TTL que depois foi usado para construir a ontologia.
    Por fim, a informação gerada foi usada para povoar a ontologia criada inicialmente.
