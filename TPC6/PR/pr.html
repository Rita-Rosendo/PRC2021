    • Abstract:

    Foi-nos proposta a realização de um trabalho de casa que tem como objetivo criar uma App em Vue para consumir a API de dados criada no tpc anterior, o TPC5.
    Em primeiro lugar, fez-se a listagem de publicações. Como, neste caso, foi-nos dito que os elementos que deveriam ser listados eram o id, o tipo, o título e o ano de pulblicação, foi necessário modificar a query do GET das publicações na API de dados.
    De seguida, fez-se a listagem dos autores. Novamente, como os elementos que deveriam ser listados são o id, o nome e o número de publicações, a query relativa ao GET dos autores no lado da API de dados foi modificada.
    Posteriormente, implementou-se a opção de ao clicar no ID de uma publicação, quando se está na página da lista de publicações, ir para a consulta da publicação onde a informação acerca da mesma é listada.
    Por fim, implementou-se a opção de ao clicar no número de publicações de um autor, quando se está na página da lista de autores, ir para a consulta do autor onde a informação acerca do mesmo é listada. Para além disso, é também possível clicar nas publicações que o mesmo fez e visualizar a informação acerca dessa publicação.
