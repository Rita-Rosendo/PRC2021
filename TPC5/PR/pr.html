    • Abstract:

    Foi-nos proposta a realização de um trabalho de casa que tem como objetivo a criar uma API de dados com as funcionaliadades de CRUD sobre o modelo criado.
    Em primeiro lugar, foi necessário analisar o dataset jcrpubs.xml. Para tal, recorreu-se à ferramenta Oxygen XML Editor onde se gerou o ficheiro DTD do ficheiro XML fornecido.
    Em segundo lugar, recorreu-se ao Protégé para construir uma parte estrutural inicial da ontologia baseada no dataset dado. Foi também necessário criar alguns indivíduos para servirem de modelo para o passo seguinte.
    Em terceiro lugar, recorreu-se ao Oxygen XML Editor onde se criou uma stylesheet XSLT necessária para a povoação da ontologia com os indivíduos do dataset.
    Em quarto lugar, carregou-se a ontologia no Ontobud criando um repositório novo denominado A84475-TP5 e importou-se o ficheiro gerado no ponto anterior.
    Por fim, criou-se uma API de dados e implementou-se as seguintes rotas: GET /pubs, GET /pubs/{id}, GET /authors, GET /authors/{id}.
