    • Abstract:

    Foi-nos proposta a realização de um trabalho de casa que tem como objetivo criar uma Ontologia a partir de um dataset em json disponibilizado na aula.
    Primeiramente, analisou-se o dataset em json "mapa_virtual.json" e criou-se uma ontologia estrutural no Protégé.
    De seguida, usando a linguagem de programação Python, gerou-se ficheiros em ttl com os indivíduos necessários para povoar a ontologia inicialmente criada.
    Por fim, juntou-se ao ficheiro ttl que contém a ontologia estrutural inicial, a informação contida nos ficheiros criados com o script em python povoando-se assim a ontologia.
