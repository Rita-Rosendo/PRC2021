    • Abstract:

    Foi-nos proposta a realização de um trabalho de casa que tem como objetivo gerar um dataset num TTL seguindo o TTL criado na aula.
    Este TPC divide-se em duas partes. A primeira é gerar um dataset em JSON de acordo com os requisitos e a segunda é criar um conversor, numa linguagem à nossa escola, de JSON para TTL de acordo com a ontologia definida na aula.
    Primeiramente, o dataset em JSON foi criado. Para tal, foi usado um gerador de datasets em JSON online, o "JSON Generator". O dataset criado seguiu os seguintes requisitos : ter 4 UCs, ter 4 Docentes e ter 200 alunos.
    De seguida, recorrendo à linguagem de programação Java, a conversão do dataset em JSON num TTL foi feita.

  • Project Record - XSL
  • Project Record - HTML
