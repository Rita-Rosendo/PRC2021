    • Abstract:

    Foi-nos proposta a realização de um trabalho de casa que tem como objetivo o desenvolvimento de um Front-End Web para mostrar a informação que está presente no GraphDB. Para tal, foi utilizada a framework Express.
    Na primeira página encontra-se a lista de repositórios disponíveis assim como alguma informação acerca dos mesmos. Em cada repositório, no campo do ID, é possível clicar e ser direcionado para a página desse repositório.
    Na página de cada repositório encontra-se a lista das suas classes. E, tal como acontece com cada repositório, ao clicar numa classe, o utilizador é dirigido para a página dos indivíduos da classe.
    Por sua vez, na página da lista dos indivíduos da classe, sempre que o utilizador clica num indivíduo este é direcionado para a página do invidíduo em questão onde se pode observar as propriedades do indivíduo clicado.
