    Foi-nos proposta a realização de um trabalho de casa que tem como objetivo a criação de um servidor NodeJS que suporta uma interface para a execução de queries em SPARQL.

    Por fim, foi desenvolvida uma app Web com os seguintos requisitos: ter uma Página Inicial, uma para a lista de elementos da tabela em que um dos campos quando clicado mostra a informação acerca de um determinado elemento, outra página para os grupos e outra para os períodos.
